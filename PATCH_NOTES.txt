1. 밈 코인 발행 시스템 DEX 스타일 전환 🔄
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

▶ 발행 비용 대폭 인하 (플래닛어스 경제에 최적화)
  기존: 4,500 ECC (약 45,630 골드) - 최고 무기의 8%
  변경: 최소 550 ECC (약 5,577 골드) - 최고 무기의 1%
  절감: 91% 비용 절감!

▶ 사용자 정의 초기 자본 시스템
  - 최소: 500 ECC 이상
  - 최대: 제한 없음 (발행자 책임)
  - 수수료: 초기 자본의 10% (동적 계산)

  예시:
  ┌─────────────────┬──────────────┬──────────────┬─────────────┐
  │  초기 자본      │  수수료      │  총 필요     │  골드 환산  │
  ├─────────────────┼──────────────┼──────────────┼─────────────┤
  │  500 ECC        │  50 ECC      │  550 ECC     │  5,577 골드 │
  │  1,000 ECC      │  100 ECC     │  1,100 ECC   │  11,154 골드│
  │  5,000 ECC      │  500 ECC     │  5,500 ECC   │  55,770 골드│
  │  10,000 ECC     │  1,000 ECC   │  11,000 ECC  │ 111,540 골드│
  └─────────────────┴──────────────┴──────────────┴─────────────┘

▶ 발행량 자유 설정
  - 기존: 최소 4,000개 이상 강제
  - 변경: 제한 없음 (자유 설정)

▶ DEX 스타일 가격 계산
  공식: 초기 가격 = 초기 자본 / 발행량

  예시:
  - 500 ECC, 10,000개 발행 → 0.05 ECC/코인
  - 5,000 ECC, 10,000개 발행 → 0.5 ECC/코인 (10배!)

2. 러그풀 방지 시스템 구현 🔒
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

▶ 제작자 보유율 10% 초과 시 자동 경고
  - 실시간 보유율 계산 및 표시
  - 빨간색 경고 메시지 표시
  - 신청 전 확인 다이얼로그
  - 서버 로그에 러그풀 위험 기록

▶ 보유율 계산 공식
  보유율 = (제작자 보유 ECC / 코인 가격) / 전체 발행량 × 100

  예시:
  - 발행량: 10,000개
  - 초기 가격: 0.5 ECC/코인
  - 제작자 보유: 1,000 ECC
  - 보유 코인: 1,000 / 0.5 = 2,000개
  - 보유율: 2,000 / 10,000 × 100 = 20% ⚠️ 경고!

================================================================================
                          📊 경제 시스템 변화
================================================================================

▶ 발행 진입 장벽 대폭 완화
  기존: 45,630 골드 (최고 무기의 8%)
  변경: 5,577 골드 (최고 무기의 1%)

  → 일반 플레이어도 밈 코인 발행 가능!

▶ 유동성 풀 시스템
  - 발행자가 원하는 만큼 자본 투입
  - 투입 자본 = 초기 유동성
  - 자본이 많을수록 초기 가격 상승

▶ 가격 형성 메커니즘
  초기 가격 = 투입 자본 / 발행량

  발행자의 선택:
  1. 적은 자본 + 많은 발행량 = 저가 코인
  2. 많은 자본 + 적은 발행량 = 고가 코인
